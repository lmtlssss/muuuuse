Muuuuse

three-seat tmux relay for tight local CLI loops

Muuuuse pairs two local listening seats and one controller seat, then relays only final answers between them.

It is tiny on purpose. Local machine only. Fast handoff. No remote control.

tmuxthree seatsfinal-answer relaylocal only
muuuuse 1
muuuuse 2
muuuuse 3 "Start by proposing the first concrete repo task."
seat model Seats 1 and 2 listen. Seat 3 pairs them and can drop a one-time kickoff prompt.
script mode muuuuse script and muuuuse live let a seat toggle between deterministic loops and a live CLI.
hard boundary Remote control is intentionally out of scope. Use CODEMAN or codemansbot for that.
